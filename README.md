# Car Price Predictor

A machine learning web application built with Streamlit that predicts car prices based on vehicle specifications.

## Features

- Car price prediction using machine learning
- Random Forest and Gradient Boosting models
- Interactive Streamlit interface
- Exploratory Data Analysis (EDA)
- Model metrics and explanations

## Performance

- Cross-Validated R²: 74%
- MAPE: 13%

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
